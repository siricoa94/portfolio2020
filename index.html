Fun With Forums

Fun With Forums is a note taking application that lets you create, update, and delete notes. When first appearing on the page, users are prompted to click a button for log in where they can either sign up as a new user or sign in. Upon login you are brought to the main page which displays two forum fields, one for a post title and another for the content of the post. Once a user submits a post it will be appended to the chalk board below. Users can edit the body of their posts or delete them if they so choose to do so.

Geting Started

In order to run this application you will need to utilize the following technologies.
  • HTML
  • CSS
  • Javascript/Jquery
  • Node.js/Express
  • MySql
  • Firebase

Found Sound

Found Sound is the perfrect application for anyone who would like to sample some music by an artist. Simply type into the search bar an artist, song, or album name and Found Sound will return you a list of samples based off the search phrase that was provided. Having a simple, yet elegant design, sound found is easy to navigate and use for any user.

Geting Started

In order to run this application you will need to utilize the following technologies.
  • HTML
  • CSS
  • Javascript/Jquery
  • Node.js/Express (optional for hosting/deployment)

Rico Depot

Rico Depot is a marketplace/game application that works as follows. Upon signing up a user is given 10,000 credits and is allowed to purchase products listed on the products page. When a user buys a product, their purchase history will be appened to their homepage. Users can also view how their spending habits score up against other users on the leaderboard page.
