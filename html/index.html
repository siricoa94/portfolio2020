Fun With Forums

Fun With Forums is a note taking application that lets you create, update, and delete notes. This application is built using HTML, CSS, Javascript, Node.js/express, firebase, and MySql.

Found Sound

Found Sound is a simple yet effective application for delivering sound samples. This application utilizes Jquery to preform an ajax call to retrieve data from the Deezer API and Rapid API to generate the key.

Rico Depot

Rico Depot is a "mock" online marketplace where users can sign up and purchase products using credits. Users can also update their profiles and see their purchase histories. Furthermore, there is a leaderboard that displays the "Top Spender" on the website!
